The admin dashboard is the central control panel for institution-wide management. After logging in as an administrator, you are directed to `/admin`, where the `AdminOverview` component renders a live snapshot of the institution's current state — including total students and teachers, active courses, pending approvals, and recent system activity. All statistics are fetched from `GET /api/admin/dashboard`.

## Dashboard statistics

The overview section surfaces four headline metrics drawn from the dashboard API response:

| Metric | API field |
|---|---|
| Total students | `total_students` |
| Faculty members | `faculty_count` |
| Active courses | `active_courses` |
| Departments | `departments_count` |

<Note>
  Statistics refresh each time you load the dashboard. The `AdminOverview` component also shows percentage changes compared to the previous month alongside each figure.
</Note>

## Recent activity feed

Beneath the statistics grid, the dashboard displays a chronological activity feed showing the latest events across the system — new registration requests, completed insurance verifications, and batch enrollment approvals. This feed is read-only and updates each time you load the page.

<Tip>
  Use the "View All" button in the Recent Enrollments table to jump directly to the full enrollment management interface at `/admin/enrollments`.
</Tip>
